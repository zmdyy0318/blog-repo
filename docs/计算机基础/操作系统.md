# 操作系统
是一种软件，与计算机硬件进行通信，并为程序执行提供环境。抽象了底层硬件的细节，使开发人员能专注于软件逻辑，而不是特定硬件。

## 操作系统分类
主流操作系统有两个：类UNIX和Windows。

|操作系统|系列|说明|
|---|---|---|
|Ubuntu|类UNIX|基于GNU/Linux|
|Android|类UNIX|使用Linux内核+安卓生态|
|iOS|类UNIX|基于Darwin|
|macOS|类UNIX|基于Darwin|
|Windows 10|Windows|基于Windows NT|

## 用户模式和内核模式
* 用户模式的代码以进程的方式运行在操作系统中，每个进程有唯一的进程ID，不能直接访问硬件。
* 用户模式进程内通过虚拟地址访问物理内存空间，不同进程的虚拟地址不能相互访问。
* 用户模式的代码可以通过调用系统API切换到内核模式，操作系统/物理硬件。
* 内核模式代码通过虚拟地址访问物理内存空间，可以访问整个内核地址。
* 内核模式代码对整个操作系统、所有硬件有完整的访问权限。
* 用户模式、内核模式均以线程作为最小单位运行代码。
